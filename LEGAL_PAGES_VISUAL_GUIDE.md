## ✨ Animasi & Interaksi

### **Animations**
```javascript
Badge:      fade-in (0s)
Heading:    slide-up (0.1s delay)
Content:    scale-in (0.2s delay)
Background: pulse animation (infinite)
```

### **Hover Effects**
```css
.glass-strong → .glass (lighter effect)
.glass → .glass-strong (stronger effect)
text-violet-400 → text-violet-300 (email links)
scale-100 → scale-105 (subtle zoom)
```

## 📱 Responsiveness

### **Mobile (< 768px)**
- Single column layout
- Stacked grid cards
- Smaller heading (5xl → 5xl)
- Reduced padding
- Touch-friendly spacing

### **Tablet (768px - 1024px)**
- 2-column grid for cards
- Medium heading (6xl)
- Balanced spacing

### **Desktop (> 1024px)**
- Full 2-column grid layouts
- Large heading (7xl)
- Maximum spacing
- Optimal reading width (max-w-4xl)

## 💡 Tips Customization

### Update Konten
Edit file:
- `src/views/terms-of-service.ejs`
- `src/views/privacy-policy.ejs`

### Update Warna
Ubah gradient badges di setiap section:
```html
<div class="bg-gradient-to-br from-violet-600 to-purple-600">
     ↑ Ubah warna sesuai kebutuhan
```

### Update Contact Info
Edit di section terakhir (Contact Us/Contact Information):
```html
<p>Jln Pendidikan Pringgasela...</p>
<a href="mailto:legal@pixelnest.id">...</a>
```
