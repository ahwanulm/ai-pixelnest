## 🛠️ HOW TO USE

### Step 1: Set Image Pricing
```
1. Go to: http://localhost:5005/admin/pricing-settings
2. Scroll to "IMAGE MODELS" section
3. Adjust slider: Set profit margin (recommended: 20%)
4. Set minimum credits: 0.5 (for cheap models)
5. Set cheap threshold: $0.01
6. Check live preview
7. Click "Save IMAGE Pricing"
```

### Step 2: Set Video Pricing
```
1. Scroll to "VIDEO MODELS" section
2. Adjust slider: Set profit margin (recommended: 25%)
3. Set minimum credits: 1.0 (for cheap models)
4. Set cheap threshold: $0.05
5. Check live preview
6. Click "Save VIDEO Pricing"
```

### Step 3: Calculate Projections
```
1. Scroll to "Revenue & Profit Projections"
2. Input your estimates:
   - Monthly Generations: 1000 (how many?)
   - Avg Credits per Gen: 2.5 (typical usage)
   - Credit Price (USD): $0.05 (your selling price)
3. See real-time projections:
   - Monthly Revenue
   - FAL.AI Costs
   - Net Profit
   - Yearly Projection
```

## ⚠️ IMPORTANT NOTES

### 1. **Credit Price Adjustment**
```
Current: $0.05/credit (default)

⚠️ Too low for video models!

Recommendation:
  - IMAGE: $0.05/credit ✅
  - VIDEO: $0.08-0.10/credit ✅
  
Or use SEPARATE credit types:
  - IMAGE Credits: $0.05
  - VIDEO Credits: $0.10
```

### 2. **Cheap Threshold**
```
Purpose: Model murah tetap profitable

IMAGE < $0.01 → Use 0.5 credits (minimum)
VIDEO < $0.05 → Use 1.0 credits (minimum)

Example:
  FAL $0.0050 (very cheap)
  → Normal: 0.12 credits (loss!)
  → With threshold: 0.5 credits ✅ (profit!)
```

### 3. **Rounding Logic**
```
Always round to nearest 0.5:
  - 1.23 → 1.0
  - 1.48 → 1.5
  - 1.75 → 2.0
  - 2.13 → 2.0

Why?
  ✅ Clean pricing
  ✅ Easy to understand
  ✅ No odd numbers
```

## 📈 MONITORING & OPTIMIZATION

### Track These Metrics:
```
1. Actual vs Projected Revenue
2. Average Credits per Generation
3. Most Used Models (high/low cost)
4. User Retention (price sensitive?)
5. Profit Margin Trends
```

### Optimize:
```

- Low volume, high profit? → Decrease margin (attract users)
- Too many cheap models? → Adjust cheap threshold
- Loss on some models? → Increase minimum credits
```
